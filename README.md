# Tratamento e Análise de Dados com Pandas

Projeto acadêmico com objetivo de aplicar técnicas de tratamento e análise de dados utilizando a biblioteca Pandas.

## Dataset
Conjunto de transações bancárias simuladas contendo:
- tipo de transação
- valor
- data
- contas de origem e destino

## Atividades realizadas
- Exploração e descrição dos dados
- Tratamento de valores nulos e tipos
- Análise exploratória
- Respostas a 5 perguntas utilizando filtros e agrupamentos

## Tecnologias
- Python
- Pandas
- Jupyter Notebook
